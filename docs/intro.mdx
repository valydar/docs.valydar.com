# Valydar Identity Platform

Valydar provides a single, unified API for identity verification — document verification, biometrics (face matching + liveness), KYC/AML screening, KYB, and fraud detection.

## One Platform, Every Check

| Product | Description |
|---|---|
| **Verify** | Government-issued ID verification — 190+ countries, 1,500+ document types |
| **Face** | Face matching with active and passive liveness detection |
| **KYC** | Customer onboarding and compliance workflows |
| **AML** | Sanctions, PEP, and watchlist screening |
| **KYB** | Business verification and company onboarding |
| **Fraud** | Device intelligence, fraud detection, and risk scoring |

## How It Works

1. **Create a verification** — choose which checks to run (document, liveness, face match, AML)
2. **Upload evidence** — document images, selfies, or both
3. **Get results** — the platform runs the checks and returns results synchronously or via webhook
